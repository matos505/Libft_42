# Libft

## Description
Libft is the first library project of the 42 curriculum. The goal is to recreate a set of standard C library functions and develop a personal library that can be reused in future projects.

## Instructions
To compile the library:
This command generates the static library `libft.a`.
```bash
make
```
To remove all object files.
```bash
make clean
```
To remove all object files and the library.
```bash
make fclean
```
To remove all generated files and recompile the library:
```bash
make re
```
